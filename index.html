University Form Project

A PHP based web application used by the staff of Stamford International University. The web application offers a digital version of the make-up class form used by the lecturers in the University. Prior to the introduction of the website, this process was done with pen and paper, involving scanning and emailing to get approved by multiple parties. This web application digitized the form creation, and approval process, as well as implementing an automated notification system to ensure the right party is alerted when a form is pending on their approval.
